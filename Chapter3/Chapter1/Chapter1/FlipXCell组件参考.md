# FlipXCell组件参考

**FlipXCell**组件能够获取让当前节点水平翻转的缓动。
![](C:\Users\你失散的亲生父亲\Documents\tweenerGitBook\Sources\flipx.png)

## 选项

| 选项           | 说明                         |
| -------------- | ---------------------------- |
| 加载完成后播放 | 生命周期onLoad时自动播放缓动 |

## 属性

| 属性         | 说明           |
| ------------ | -------------- |
| 缓动组件标签 | 用于标识的标签 |
