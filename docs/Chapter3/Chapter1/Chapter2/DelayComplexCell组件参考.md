# DelayComplexCell组件参考

**DelayComplexCell**组件能够获取目标为当前节点的**延迟**缓动。

![image-20201108235102469](C:\Users\你失散的亲生父亲\Documents\tweenerGitBook\Sources\delayComplexCell.png)

## 组件的选项

| 选项           | 说明                         |
| -------------- | ---------------------------- |
| 加载完成后播放 | 生命周期onLoad时自动播放缓动 |

## 组件的属性

| 属性         | 说明             |
| ------------ | ---------------- |
| 缓动组件标签 | 用于标识的标签   |
| Asset        | 数据集的json资源 |



## 延迟缓动数据的属性

| 属性     | 说明            |
| -------- | --------------- |
| 名称     | 缓动数据的名称  |
| ID(只读) | 缓动数据的ID    |
| 等待时间 | Delay缓动的时间 |
