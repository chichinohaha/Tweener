# ShowCell组件参考

**ShowCell**组件能够获取让当前节点展示渲染的缓动。

![image-20201108125624163](https://raw.githubusercontent.com/chichinohaha/Tweener/gh-pages/docs/Sources/show.png)

# 

## 选项

| 选项           | 说明                         |
| -------------- | ---------------------------- |
| 加载完成后播放 | 生命周期onLoad时自动播放缓动 |

## 属性

| 属性         | 说明           |
| ------------ | -------------- |
| 缓动组件标签 | 用于标识的标签 |
